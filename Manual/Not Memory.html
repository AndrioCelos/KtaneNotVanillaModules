On the Subject of Not Memory

Memory is a annoying module but so is everything else on the bomb itself, so does it matter?

Press the correct button to disarm the module.
Only the one correct button need to be pressed to disarm the module.
Pressing an incorrect button will register a strike and reset the module.
Button positions are numbered from left to right.

No lit LEDs:

  • If the display is 1, press the button at the position of the second button’s label.
  • Otherwise, if the first button is a 4, press the button with the same label as the display.
  • Otherwise, if the last digit of the Serial Number is odd, press the button labeled ‘1’.
  • Otherwise, press the button at the position of the display.

1 lit LED:

  • If there are no batteries on the bomb, press the button labeled ‘4’.
  • Otherwise, if the third button is a 1, press the button in the first position.
  • Otherwise, if the display is 3, press the button at the position of the third button’s label.
  • Otherwise, if the second button’s label matches the display, press the second button.
  • Otherwise, press the button labled ‘1’.

2 lit LEDs:

  • If the display is 4, press the button with the same label at the position of the button labeled ‘4’.
  • Otherwise, if the second button is labeled ‘3’, press the button on the first position.
  • Otherwise, if the fourth button’s label matches the display, press the button at the position of the display.
  • Otherwise, press the button on the second position.

3 lit LEDs:

  • If the bomb has a Parallel port, press the button on the first position.
  • Otherwise, if the fourth button is labeled ‘1’, press the button on the second position.
  • Otherwise, if the display is 2, press the button with the same label as the position of the button labeled ‘3’.
  • Otherwise, if the first button’s label matches the display, press the button at the position of the display.
  • Otherwise, press the button labeled ‘2’.

4 lit LEDs:

  • If the display is 3, press the button at the position of the second button’s label.
  • Otherwise, if the second button is labeled ‘1’, press the button on thr first position.
  • Otherwise, if the first button’s label matches the display, press the button on the second position.
  • Otherwise, if the bomb has more than 3 indicators total, press the button labeled ‘3’.
  • Otherwise, if the fourth button is labeled ‘4’, press the fourth button.
  • Otherwise, press the button labeled ‘2’.

5 lit LEDs:

  • If the bomb has no ports at all, press the button labeled ‘2’.
  • Otherwise, if the display is 4, press the button at the position of the fourth button’s label.
  • Otherwise, if the first button is labeled ‘3’, press the button on the third position.
  • Otherwise, if the third button’s label matches the display, press the button at the position of the display.
  • Otherwise, if the second button is NOT labeled ‘2’, press the button labeled ‘2’.
  • Otherwise, press the button on the second position.
